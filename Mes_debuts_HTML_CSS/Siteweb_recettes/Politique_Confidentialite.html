Politique de confidentialité

Cette déclaration de confidentialité couvre les informations pratiques de simplementbon.com. En consultant le site web simplementbon.com, vous acceptez les conditions de ce document de politique de confidentialité.

1.Informations personnelles collectées

Informations que vous fournissez

Cécilia Hiribarne collecte les informations que vous fournissez lors de toute demande via le formulaire en ligne sur la page Contact. En fournissant ces informations, vous consentez que vos informations soient collectées et utilisées, uniquement dans le but de traiter votre demande. Ces informations incluent votre nom, prénom, téléphone et adresse email.

Informations de navigation

En naviguant sur simplementbon.com, Cécilia Hiribarne collecte des informations via l’usage de cookies : cela inclut des informations standards telles que votre navigateur web, votre adresse IP et les actions que vous faites sur simplementbon.com ou des balises web. Vous pouvez bloquer tous les cookies qui ont été sauvegardés sur votre ordinateur en suivant les instructions de votre navigateur pour les désactiver. Si vous bloquez les cookies, il pourrait être impossible d’utiliser certains services.

2.Utilisation des informations collectées


Aucun usage marketing (newsletter) n’est fait par Cécilia Hiribarne des e-mails collectés via le formulaire en ligne.

3.Paramétrage des cookies


Un cookie est un fichier texte contenant une petite quantité d’informations qui est installé sur votre ordinateur ou votre périphérique mobile lors de votre première visite sur un site Internet. Les cookies remplissent un certain nombre de fonctions : ils vous permettent, par exemple, de naviguer de manière fluide entre les pages, retiennent vos préférences, et permettent en général d’améliorer l’expérience de l’utilisateur. Vous pouvez à tout moment modifier vos paramètres en configurant la gestion des cookies sur notre site web, avec la possibilité de configurer chaque service.

4.Modification de politique de confidentialité


Cécilia Hiribarne se réserve le droit de changer la présente Politique de confidentialité en vous notifiant via ce site web.

5.Contact


Toute question concernant cette Politique de confidentialité ou toute demande d’accès à vos données personnelles doit être directement adressée par e-mail via la page contact.
